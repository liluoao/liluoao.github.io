发布包到Packagist

作为 PHP 使用者，平常会接触到 Composer 与很多包。如何发布一个自己的包让别人使用呢，下面介绍下实现步骤。

Composer 介绍

Composer 是 PHP 的依赖管理工具，它为你解决这类问题：

  1. 你有一个项目依赖于若干个库
  2. 其中一些库依赖于其他库
  3. 你声明你所依赖的东西
  4. 找出哪个版本的包需要安装，并安装它们（将它们下载到你的项目中）

Composer 的安装方式见文档

composer.json

要想使用 Composer，就必须要一个 composer.json 文件，该文件包含了项目的依赖和其它的一些元数据。
