Kong框架的AdminAPI

介绍

Kong 附带了一个用于管理的内部 RESTful Admin API。可以将对 Admin API 的请求发送到集群中的任何节点，并且 Kong 将使所有节点上的配置保持一致。admin_listen 默认端口配置

  • 8001 是 Admin API 侦听的默认端口。
  • 8444 是 HTTPS 的默认端口。

此 API 专为内部使用而设置，在设置 Kong 环境时应谨慎，避免把 API 暴露给外部。
