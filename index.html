PSR2

PHPStorm 的 CodeStyle

Code Style 是你格式化代码时依据的规范。

当你修改了一段代码，想要 Commit 的时候，建议先选中你的代码使用 CTRL+ALT+L 格式化一下（非常熟悉后写出来的就是格式了）。
